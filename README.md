# iflixUserProfiling
The repository contains a dataset related to iflix user profiles. It consists with the values in following columns. Values are in text, numbers, currency formats.
* MSISDN_VOICE	
* AGE	
* DISTRICT_NAME	
* CONTACT_GENDER	
* SEGMENTATION_CODE	
* ACCOUNT_TYPE	
* AVERAGE_MONTHLY_BILL_AMOUNT	
* PRE_AVG_REFILL_SIX	
* DEVICE_CATEGORY	
* DEVICE_BRAND	
* DEVICE_NAME	
* HSPA	
* ACCOUNT_STATUS	
* PACKAGE_CODE	
* GEOGRAPHY	
* DATA_USAGE	
* VOICE_ARPU	
* DATA_ARPU
